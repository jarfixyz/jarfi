# Jarfi Documentation

**Jarfi** is an onchain savings jar app built on Solana. Create a jar, set a goal, share a link — anyone can contribute by card. Funds earn yield automatically.

- **Website:** https://jarfi.xyz
- **Dashboard:** https://jarfi.xyz/dashboard
- **Network:** Solana (devnet during beta, mainnet at launch)
