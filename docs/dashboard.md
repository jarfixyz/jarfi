# Dashboard Guide

The dashboard at [jarfi.xyz/dashboard](https://jarfi.xyz/dashboard) is your main control center.

---

## Sidebar

| Item | Description |
|------|-------------|
| **jarfi.xyz** (top) | Link back to the landing page |
| **+ New jar** | Opens the jar creation wizard |
| **My jars** | Your jar list and portfolio overview |
| **Activity** | Savings timeline and analytics |
| **People** | Contributors across all your jars |
| **Demo** | Live demo with example jars |
| Wallet address (bottom) | Your connected wallet |

---

## My Jars tab

### Overview (hero section)

Shows a summary of your portfolio:
- **Total saved** — sum of all jar balances
- **Monthly yield** — estimated monthly earnings at current APY
- **Est. at goal** — projected value at unlock dates
- **Contributors** — unique addresses that have contributed

### Sort & Filter

Use the sort dropdown above the jar list to order by:
- **Recent** — order of creation
- **Amount** — highest balance first
- **Progress** — closest to goal first
- **Unlock date** — soonest unlocking first

### Jar Cards

Each card shows:
- Jar name, emoji, and type badge
- Current balance and goal progress bar
- Projected value at unlock (green, with ?)
- **+ Add funds** — opens Transak to deposit by card
- **Share** — copies the gift link

Click any card to open the **Jar Detail Panel**.

## Jar Detail Panel

The full detail view for a single jar.

| Section | Description |
|---------|-------------|
| **Current balance** | How much is in the jar right now |
| **Est. at unlock** | Projected balance including yield |
| **Progress bar** | % of goal reached |
| **Staking yield** | Current APY, monthly earnings, protocol |
| **Forecast** | How the balance grows over time |
| **Gift link** | Share URL + WhatsApp / Twitter / Telegram buttons |
| **Schedule** | Set up monthly reminders |
| **Contributions** | List of who contributed and how much |
| **Family approval** | Co-signer setup (if enabled) |
| **Break jar** | Withdraw all funds (only available when unlock conditions are met) |

## Breaking a jar

To withdraw funds:
1. Open the jar detail
2. Click **Break jar & withdraw funds**
3. Confirm in the modal
4. Approve two transactions in your wallet (unlock + withdraw)
5. USDC arrives in your wallet within seconds

The break button is **grey and disabled** when the jar is still locked (date hasn't arrived or goal not reached). For Shared jars, the button is always available.
